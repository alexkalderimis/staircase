freq

(freq & ignored-params)

This function returns 1 regardless of inputs. Very useful for determining a count with pivot tables

full-outer-join

(full-outer-join left-coll right-coll join-fn)(full-outer-join left-coll right-coll left-join-fn right-join-fn)

This is for performing full outer joins. The join value may exist in either list. This function takes a left collection, a right collection, and at least one join function. If only one join function is provided, it is used on both the left & right hand sides.

full-outer-style

(full-outer-style left-keys right-keys)

inner-join

(inner-join left-coll right-coll join-fn)(inner-join left-coll right-coll left-join-fn right-join-fn)

This is for performing inner joins. The join value must exist in both lists. This function takes a left collection, a right collection, and at least one join function. If only one join function is provided, it is used on both the left & right hand sides.

inner-style

(inner-style left-keys right-keys)

join-worker

(join-worker join-style left-coll right-coll join-fn)(join-worker join-style left-coll right-coll left-join-fn right-join-fn)

This is an internal method to be used in each join function.

left-outer-join

(left-outer-join left-coll right-coll join-fn)(left-outer-join left-coll right-coll left-join-fn right-join-fn)

This is for performing left outer joins. The join value must exist in the left hand list. This function takes a left collection, a right collection, and at least one join function. If only one join function is provided, it is used on both the left & right hand sides.

left-outer-style

(left-outer-style left-keys right-keys)

map-vals

(map-vals f coll)

multi-pivot

(multi-pivot coll grouping-fn & fns)

This function is designed to reudce a list of tuples to a single map. It is supposed to take a list of alternating mapping and reduction functions. The resulting values are stored in a vector.

natural-join

(natural-join left-coll right-coll)

Performs the natural join. If there are no keys that intersect, the join is not performed.

right-outer-join

(right-outer-join left-coll right-coll join-fn)(right-outer-join left-coll right-coll left-join-fn right-join-fn)

This is for performing right outer joins. The join value must exist in the right hand list. This function takes a left collection, a right collection, and at least one join function. If only one join function is provided, it is used on both the left & right hand sides.

right-outer-style

(right-outer-style left-keys right-keys)
